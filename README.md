# FocusForge

A lightweight, gamified productivity app that helps you start, focus, and follow through — one tiny win at a time.

## What it is

FocusForge is a mobile-first, dark-mode web app that combines tasks, habits, goals, a focus timer, and a rewards system into a single simple interface. All data is stored locally in the browser — no account required.

## How to run

```bash
npm install
npm run dev
```

Open [http://localhost:5000](http://localhost:5000) in your browser.

To build for production:

```bash
npm run build
```

## MVP features

- **Today tab** — Main goal, Tiny Start action, Top 3 tasks, Next Reward preview, and Start New Day reset
- **Focus tab** — 5, 10, and 25-minute focus timer with a progress ring, optional sound and haptic feedback
- **Goals tab** — Add goals with Why It Matters, Tiny Next Step, Minimum Win, and Backup Plan; Break It Down action steps
- **Habits tab** — Daily habit tracking with Full win / Partial win / Skip and streak counters
- **Rewards tab** — Fully customizable reward list; claim rewards with earned points
- **Points system** — Earn points by completing tasks, habits, and focus sessions; spend them on rewards
- **Persistence** — Everything saved to localStorage; no server needed
